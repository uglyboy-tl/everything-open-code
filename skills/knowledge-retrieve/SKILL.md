# 知识检索

## 检索流程

### 阶段一：定位分类

```bash
cat 知识梳理/index.md
```

- 根据关键词（工具、数学、系统等）确定目标分类
- 注意"跨域关联"区块可能提供线索

### 阶段二：深入分类

```bash
cat 知识梳理/<分类>/index.md
```

- 查看分类下的所有笔记和子分类
- 通过摘要判断相关性，记录笔记路径

### 阶段三：提取内容

```bash
cat 知识梳理/<分类>/<笔记>.md
```

- 读取笔记完整内容
- 检查底部"参考资料"中的 `[[双链]]` 关联
- 相关则递归读取关联笔记

## 辅助搜索

索引浏览无法定位时：

```bash
rg "关键词" 知识梳理/ -t md -l
```

获取文件列表后选择性读取。

## 检索策略

1. 从粗到细：总索引 → 分类索引 → 具体笔记
2. 关键词优先：明确术语时直接搜索
3. 追踪双链：`[[链接]]` 是知识网络线索
4. 摘要先行：利用索引摘要避免盲目读取

## 输出原则

- 直接返回找到的内容或摘录
- 找不到时说明原因即可
