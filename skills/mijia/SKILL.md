## 二、命令速查

```bash
# API 控制
uvx mijiaAPI get --did <DID> --prop_name <属性>              # 读取
uvx mijiaAPI set --did <DID> --prop_name <属性> --value <值>  # 设置
uvx mijiaAPI --list_consumable_items                          # 耗材

# 语音控制（小爱音箱/小爱触屏）
uv run scripts/speaker.py "指令"                              # 执行指令并播报
uv run scripts/speaker.py "指令" --quiet                      # 静默执行
uv run scripts/speaker.py "文本" --tts                        # TTS 播报文本
uv run scripts/speaker.py "文本" --tts --did <DID>            # 指定设备 TTS 播报
```

### 常用属性

| 设备类型 | 属性 | 值域 |
|----------|------|------|
| 灯具 | `on`, `brightness`(1-100), `color-temperature`, `mode` | 台灯模式: 0-6<br>羽毛灯模式: 4-12 |
| 开关 | `on`(左), `switch-on`(右), `temperature`, `relative-humidity` | - |
| 插座 | `on`, `power-consumption` | - |

### 台灯/羽毛灯模式值

**台灯** (did: 847008456): 0=自由, 1=阅读, 2=电脑, 3=温馨, 4=休闲, 5=办公, 6=娱乐

**羽毛灯** (did: 1116662145): 4=日光, 5=月光, 7=温馨, 8=影院, 9=阅读, 10=电脑, 11=会客, 12=娱乐

## 三、决策流程

```
用户请求 → 识别设备 → 查表确认控制方式 → 执行 → 反馈
              ↓
        中键控制？ ──YES──→ 语音控制
              ↓NO
         API 直接控制
```

**执行前必确认**：
1. 设备是否存在且在线（`uvx mijiaAPI -l`）
2. 属性是否支持
3. 灯带仅开关、羽毛灯开关用客厅开关左键

**执行后必反馈**：成功确认 / 失败原因+替代方案

## 四、关键场景示例

### 调节亮度（需先读取）
```bash
# 用户: "客厅灯调亮一点"
uvx mijiaAPI get --did 1116662145 --prop_name brightness  # 当前50
uvx mijiaAPI set --did 1116662145 --prop_name brightness --value 70
# 反馈: "客厅羽毛灯亮度已从50调整到70"
```

### 中键设备（API 不可达）
```bash
# 用户: "关闭主卧吸顶灯"
uv run scripts/speaker.py "关闭主卧吸顶灯" --quiet
# 反馈: "已关闭主卧吸顶灯"
```

### 查询温湿度
```bash
uvx mijiaAPI get --did 1095834661 --prop_name temperature
uvx mijiaAPI get --did 1095834661 --prop_name relative-humidity
# 反馈: "主卧当前温度25.3℃，湿度65%"
```

### 信息/媒体查询
```bash
uv run scripts/speaker.py "今天天气怎么样"      # 查询不加 --quiet
uv run scripts/speaker.py "播放周杰伦的歌"
```

### 耗材查询
```bash
uvx mijiaAPI --list_consumable_items
# 返回: 扫地机主刷、滤网、电池等耗材剩余寿命
# 注意: 部分离线设备可能报错
```

## 六、故障排查

| 问题 | 解决 |
|------|------|
| 设备无响应 | `uvx mijiaAPI -l` 确认在线；检查 did |
| 属性不支持 | `uvx mijiaAPI --get_device_info <model>` |
| 中键无效 | 必须用语音控制 |
| 认证失败 | 检查 `~/.config/mijia-api/auth.json` |
| 灯带异常 | 仅控制开关，调节属性会破坏流光模式 |

## 七、红旗警告

**禁止**：
- 调节灯带亮度/色温（破坏流光模式）
- 用 API 控制开关中键（不可达）
- 开关羽毛灯直接用羽毛灯 did（应使用客厅开关左键）

**始终**：
- 优先使用 `--did`
- 调节亮度/色温前先读取当前值
- 执行后简洁反馈
